精靈

用於封裝ggame應用程序中所有可見對象的Sprite類。


精靈
類ggame.sprite.Sprite（資產，POS =（0，0） ，edgedef =無）[源]

Sprite類結合了可視/圖形資產的概念，屏幕上的位置和行為。雖然Sprite可以按原樣使用，但它通常是子類，以給它一些所需的行為。

在子類化Sprite類時，您可以自定義初始化代碼以使用特定資產。可以添加“步驟”或“輪詢”方法來處理每幀動作（例如，檢查衝突）。App類不會自動調用步或輪詢函數，但您可以為App該類創建子類。

此外，您可能希望在自定義的sprite類中定義事件回調方法。通過自定義創建，事件處理和定期處理，您可以實現精靈對象的完全自主行為。

參數：
  • asset（asset） - 現有的圖形資產
  • pos（tuple （int ，int ）） - 可以提供精靈位置，它指定屏幕上精靈的起始（x，y）坐標。默認情況下，精靈的位置定義其左上角的位置。可以通過自定義此行為來修改此行為 center。
  • edgedef（資產） - 可以提供邊緣定義資產，其指定將用於定義精靈的邊界以用於衝突檢測的資產。如果沒有 給出edgedef資產，則使用所需資產，在圖像紋理的情況下，該資產將是矩形資產。此選項通常用於為具有透明紋理圖像背景的基於紋理的精靈定義可見圖像輪廓。
返回：

沒有。如果位置在屏幕上，精靈將顯示在瀏覽器中。

例：

from ggame.sprite import Sprite
from ggame.asset import ImageAsset, CircleAsset
from ggame.app import App

player = Sprite(
    ImageAsset("bunny.png",
    (100,100),
    CircleAsset(50))

App().run()

這將使用'player.png'圖像創建一個精靈，該圖像的左上角位於坐標（100,100）處，並且具有50像素半徑的圓形碰撞邊界。

setExtents（）[來源]

根據位置，中心，寬度，高度更新最小/最大x和y

firstImage（）[來源]

選擇並顯示此精靈使用的第一個圖像。如果資產是ImageAsset 使用多個圖像定義的，那麼這只會有用。

lastImage（）[來源]

選擇並顯示此精靈使用的最後一個圖像。如果資產是ImageAsset 使用多個圖像定義的，那麼這只會有用。

nextImage（wrap = False ）[來源]

選擇並顯示此精靈使用的下一個圖像。如果當前圖像已經是最後一張圖像，則圖像不會前進。

參數： wrap（boolean） - 如果為True，則調用 nextImage()最後一個圖像將導致 加載第一個圖像。

如果資產是ImageAsset使用多個圖像定義的，那麼這只會有用 。

prevImage（wrap = False ）[來源]

選擇並顯示此精靈使用的上一個圖像。如果當前圖像已經是第一張圖像，則圖像不會改變。

參數： wrap（boolean） - 如果為True，則調用 prevImage()第一個圖像將導致加載最後一個圖像。

如果資產是ImageAsset使用多個圖像定義的，那麼這只會有用 。

setImage（index = 0 ）[來源]

通過給出索引來選擇要顯示的圖像。

參數： index（int） - 指定要顯示的圖像的索引。值為零表示資產中的第一個圖像。

這相當於index直接設置屬性。

如果資產是ImageAsset使用多個圖像定義的，那麼這只會有用 。

index

這是此精靈可用圖像列表的整數索引。

width

這是一個表示精靈的顯示寬度的整數。為寬度分配值將水平縮放圖像。

height

這是一個表示精靈顯示高度的整數。為高度指定值將垂直縮放圖像。

x

這表示屏幕上精靈的x坐標。為此屬性賦值將水平移動精靈。

y

這表示屏幕上精靈的y坐標。為此屬性賦值將垂直移動精靈。

position

元組指示精靈在屏幕上的位置。

fxcenter

這表示精靈“中心”的水平位置，作為0.0到1.0之間的浮點數。值0.0表示精靈的x坐標指的是左手邊緣。值1.0表示其右邊緣。可以指定其間的任何值。可以將值分配給此屬性。

fycenter

這表示精靈“中心”的垂直位置，作為0.0到1.0之間的浮點數。值0.0表示精靈的x坐標指的是其上邊緣。值1.0表示其下邊緣。可以指定其間的任何值。可以將值分配給此屬性。

center

該屬性將精靈“center”的水平和垂直位置表示為浮點數的元組。見描述的fxcenter和fycenter更多的細節。

visible

此佈爾屬性可用於更改精靈的可見性。將~ggame.Sprite.visible設置為False將阻止精靈在屏幕上呈現。

scale

此屬性可用於更改屏幕上精靈的大小（“縮放”它）。值可以是浮點數。值為1.0表示精靈圖像將保持其原始大小。值為2.0會使其加倍，等等。

rotation

此屬性可用於更改屏幕上精靈的旋轉。值可以是浮點數。值0.0表示無旋轉。值1.0表示逆時針方向旋轉1弧度。一個弧度為180 / pi或約57.3度。

classmethod collidingCircleWithPoly（circ，poly ）[來源]

確定CircleAsset精靈是否與PolygonAsset精靈重疊。在確定兩個對像在其整個範圍中重疊之後調用此方法。

參數：
  • circ（Sprite） - 一個基於CircleAsset的精靈。
  • poly（Sprite） - 基於PolygonAsset的精靈。
返回：

如果精靈重疊則為真，否則為假。
返回類型：

布爾
collidingPolyWithPoly（obj ）[來源]

確定一對基於PolygonAsset的精靈是否重疊。在確定兩個對像在其整個範圍中重疊之後調用此方法。如果self是基於PolygonAsset的sprite，則只應調用此方法。

參數： obj（Sprite） - 一個基於PolygonAsset的精靈。
返回： 如果slef與obj重疊，則為True，否則為False。
返回類型： 布爾
collidingWith（obj ）[來源]

確定此精靈當前是否與另一個精靈對象重疊。

參數： obj（Sprite） - 對另一個Sprite對象的引用。
返回類型： 布爾
返回： 如果精靈重疊，則為真，否則為假。
collidingWithSprites（sclass = None ）[來源]

確定此精靈是否與某個類的任何其他精靈發生碰撞。

參數： sclass（class） - 一個類標識符，它是Sprite 或者是它的子類，用於標識要檢查衝突的sprite類。如果為None，則Sprite檢查從類中子類化的所有對象。
返回類型： 名單
返回： 與此sprite重疊的給定類的sprite對象的（可能為空）列表。
destroy（）[來源]

防止在碰撞檢測中顯示或檢查精靈。一旦調用它，就不能再顯示或使用精靈了。如果您只想阻止顯示精靈，請將該visible屬性設置為False。
