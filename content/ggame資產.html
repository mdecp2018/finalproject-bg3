ggame資產

ggame資產和相關對象（Color和LineStyle）是封裝和表示可顯示圖像的類。單個資產可以用於多個精靈中。可以從包含多個圖像的任何圖像（即精靈表）創建動畫資源。

框架
class ggame.asset.Frame（x，y，w，h ）[來源]

Frame是一個實用類，用於表達矩形區域的概念。

初始化參數描述框架左上角的位置以及框架的寬度和高度。單位通常 以像素為單位，但幀可以是通用的。

參數：
  • x（int） - 框架左上角的X坐標
  • y（int） - 框架左上角的Y坐標
  • w（int） - 幀的寬度
  • h（int） - 框架的高度
x

該幀左上角的X坐標。

y

該框架左上角的Y坐標。

w

框架的寬度。

h

框架的高度。

center

的中心屬性計算用於該幀的中心的坐標對（元組）。

的中心屬性，設置時，重新定義了X和ÿ框架的屬性，以使中心與分配給它的坐標（元組）一致。


顏色
class ggame.asset.Color（color，alpha ）[來源]

Color類用於表示具有透明度的顏色和/或顏色。

參數：
  • color（int） - 傳統格式的整數（通常為十六進製文字，例如0xffbb33，表示三種顏色分量，紅色，綠色和藍色）
  • alpha（float） - 透明度值，或alpha作為浮點數，範圍為0.0到1.0，其中0.0表示完全透明，1.0表示完全不透明。

例：

"""
Example of using Color class.
"""
from ggame import Color

RED = Color(0xFF0000, 1.0)

LineStyle 
class ggame.asset.LineStyle（寬度，顏色）[來源]

LineStyle類用於在繪製矩形，橢圓等幾何對象時表示線條樣式。

參數：
  • width（int） - 以像素為單位的行寬
  • color（Color） - 顏色作為有效Color實例。

例：

"""
Example of using LineStyle class.
"""
from ggame import LineStyle, Color

LINE = LineStyle(3, Color(0x00FF00, 1.0))

這定義了一個3像素寬的綠線。


預定義的顏色和線條
ggame.asset.BLACK=黑色

默認黑色

ggame.asset.WHITE=白

默認白色

ggame.asset.BLACKLINE= LineStyle（1，BLACK）

默認細黑線

ggame.asset.WHITELINE= LineStyle（1，WHITE）

默認的細白線


ImageAsset 
class ggame.asset.ImageAsset（url，frame = None，qty = 1，direction ='horizo​​ntal'，margin = 0 ）[source]

ImageAsset類將ggame連接到特定的圖像文件。

參數：
  • url（str） - 所有ImageAsset實例必須指定jpg或png圖像所在的文件名或URL。
  • frame（Frame） - 如果所需的精靈圖像僅存在於原始圖像的較小子部分中，則使用frame參數指定圖像中的區域，該參數必須是有效的 實例。Frame
  • qty = 0（int） - 如果圖像文件實際上是圖像的集合，例如所謂的精靈表，則ImageAsset類支持定義圖像列表，前提是它們在原始圖像中作為一行均勻存在間隔圖像或一列圖像。要指定此項，請在行或列中提供圖像的數量（數量）。
  • direction ='horizo​​ntal'（str） - 對於圖像精靈表，指定圖像是以“垂直”還是“水平” 排列方式。
  • margin = 0（int） - 如果圖像精靈表中的連續圖像之間存在間隙 ，則指定間隙的大小（以像素為單位）。當以這種方式使用時，frame參數必須僅定義集合中第一個圖像的區域; 假設列表中的所有後續圖像大小相同，但以邊距值分隔 。

例：

"""
Example of using ImageAsset class.
"""
from ggame import ImageAsset

IMG = ImageAsset("bunny.png")
url=無


一個字符串，表示原始文件的路徑或URL。

append（url，frame = None，qty = 1，direction ='horizo​​ntal'，margin = 0 ）[source]

從新圖像文件（或URL）附加紋理資源。此方法允許您將一組圖像構建到資源中（例如您可能需要動畫精靈），但不使用單個精靈圖像。

append方法的參數與提供給ImageAsset初始化方法的參數相同。

此方法允許您在精靈表或工作表中構建由多行或多列圖像組成的資產。

destroy（）

銷毀或取消分配資產使用的任何基礎圖形資源。在任何不再使用的資產上調用此方法。

gfx

gfx屬性表示用於表示此資產的基礎系統對象。如果此資產由多個資產組成，則第 一個資產由gfx引用。

PolygonAsset 
class ggame.asset.PolygonAsset（path，line = LineStyle（1，BLACK），fill = BLACK ）[source]

PolygonAsset是一個動態創建的“虛擬”資產，無需創建圖像文件。

注意：您不應該指定此資產的絕對屏幕坐標，因為您將使用該Sprite位置在屏幕上定位多邊形。

參數：
  • path（list） - 像素坐標元組列表。這些坐標不應該在絕對屏幕坐標中，而應該相對於所得到的所需“中心” Sprite。列表中的最終坐標對必須與第一個坐標對相同。
  • line = BLACKLINE（LineStyle） - 橢圓邊框的顏色和寬度
  • fill = BLACK（Color） - 橢圓體的顏色

例：

"""
Example of using the PolygonAsset class.
"""

from ggame import PolygonAsset, LineStyle, Color, BLACK

POLY = PolygonAsset(
    [(0, 0), (50, 50), (50, 100), (0, 0)], LineStyle(4, BLACK), Color(0x80FF00, 0.8)
)
destroy（）


銷毀或取消分配資產使用的任何基礎圖形資源。在任何不再使用的資產上調用此方法。

gfx

該GFX屬性表示底層系統對象。


LineAsset 
class ggame.asset.LineAsset（x，y，line = LineStyle（1，BLACK））[source]

LineAsset是一個即時創建的“虛擬”資產，無需創建圖像文件。LineAsset實例表示單個線段。

注意：您不應該指定此資產的絕對屏幕坐標，因為您將使用該Sprite位置在屏幕上找到您的線。線段將從像素坐標（0,0）開始，並將在下面給出的（x，y）坐標處結束。

由於LineAsset未覆蓋區域，因此只能提供LineStyle 參數（線）來指定顏色。

參數：
  • x（int） - 線端點的x坐標，以像素為單位
  • y（int） - 行終點的y坐標，以像素為單位
  • line = BLACKLINE（LineStyle） - 橢圓邊框的顏色和寬度
destroy（）

銷毀或取消分配資產使用的任何基礎圖形資源。在任何不再使用的資產上調用此方法。

delta_x=無

此屬性表示實例化期間提供的x參數。

delta_y=無

此屬性表示實例化期間提供的y參數。

gfx

該GFX屬性表示底層系統對象。


TextAsset 
class ggame.asset.TextAsset（text，** kwargs ）[來源]

TextAsset是一種即時創建的“虛擬”資產，無需創建圖像文件。TextAsset實例表示文本塊及其樣式（字體，顏色等）。

參數：
  • text（str） - 應顯示的文本
  • ** kwargs - 可選的格式和样式屬性（如下）
  • style ='20px Arial'（str） -

    文字樣式，大小和字體。例：

    'italic 20pt Helvetica'
  • width = 100（int） - 屏幕上文本塊的寬度，以像素為單位。
  • fill = BLACK（Color） - Color指定文本顏色和透明度的實例。
  • align ='left'（str） - 塊的對齊方式。“左”，“中”或“右”之一。

完整示例：

"""
Example of using TextAsset class.
"""
from ggame import TextAsset, Color

TA = TextAsset(
    "Sample Text", style="bold 40pt Arial", width=250, fill=Color(0x1122FF, 1.0)
)
